# Steam Game Recommender — SPEC.md

## 프로젝트 개요

AI 기반 Steam 게임 추천 + 할인 게임 브라우징 서비스.
유저의 Steam 라이브러리(플레이타임, 장르)와 직접 선택한 조건을 조합하여 개인화된 게임을 추천하고, 비로그인 유저에게도 할인/인기 게임 탐색 기능을 제공한다.

## 타겟 유저

| 유저 유형 | 진입 동기 |
|-----------|-----------|
| 라이트 유저 | 스트리머 영상 또는 과거 플레이 경험으로 "비슷한 게임 없나?" 탐색 |
| 할인 게임 헌터 | 눈여겨봤던 게임 할인 여부 확인, 할인 목록에서 괜찮은 게임 발굴 |

---

## 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Next.js (SSR 중심) |
| Backend | Node.js (1차) → Java/Spring 마이그레이션 (2차) |
| Database | PostgreSQL |
| AI | 경량 API 모델 (GPT-4o-mini / Claude Haiku / Gemini Flash 중 비용 비교 후 확정) |
| 상태 관리 | TanStack Query (추천 결과 캐싱 포함) |
| 인증 | Steam OpenID |
| 외부 API | Steam Web API, YouTube Data API v3 |

## 페이지 구조

### 1. 랜딩 페이지 (`/`)

비로그인/로그인 공통. 넷플릭스 스타일 레이아웃.

**구성 (위→아래):**
- **히어로 슬라이더**: 인기 할인 게임 Top 5 (자동 슬라이드, 대형 배너)
- **섹션: 압도적 긍정적 할인 게임**: 리뷰 5,000개 이상 & 압도적 긍정적~대체로 긍정적 & 현재 할인 중 (가로 스크롤 카드)
- **섹션: 인기 게임**: 할인 여부 무관, 압도적 긍정적~대체로 긍정적 리뷰 기준 (가로 스크롤 카드)
- **추가 섹션**: 장르별 / 할인율순 등

**필터링/정렬 옵션:** 장르별, 인기별, 할인 퍼센티지 높은순

### 2. AI 추천 페이지 (`/recommend`)

Steam 로그인 필수. 비로그인 시 클릭하면 Steam 로그인으로 리다이렉트.

**유저 플로우:**
1. 4가지 프리셋 선택지 입력
   - 장르: FPS, 소울류, 퍼즐 등
   - 희망 가격대: 1만원, 2만원 등
   - PC 사양: 20시리즈 / 30시리즈 / 40시리즈 / Mac
   - 원하는 플레이타임: 50시간 이내 / 100시간 이내 / 100시간 초과
2. "추천 받기" 버튼 클릭
3. 로딩 (AI 분석 중)
4. 결과 표시

**AI 입력 구조:**
- Steam 라이브러리 데이터 (소유 게임, 장르, 플레이타임) — 자동
- 유저 선택 조건 4가지 — 수동

**결과 표시:**
- **Top 3**: 근거 카드 (유저 플레이 성향 + 게임 특징 매칭 이유)
- **나머지**: AI 추천도 점수 높은순 리스트 나열

**캐싱:** TanStack Query로 캐시. 페이지 이동 후 재진입 시 캐시 결과 표시. "다시 추천받기" 시 invalidate 후 재호출. 선택지 뒤로가기 지원.

### 3. 게임 상세 페이지 (`/game/[id]`)

**필수 정보:**
- 게임 기본 정보 (제목, 설명, 장르, 출시일)
- 가격 / 할인 정보
- 리뷰 요약 (긍정 비율, 리뷰 수)
- 스크린샷
- YouTube 영상 (해당 게임 관련 조회수순 영상 또는 인기 쇼츠)
- Steam 스토어 외부 링크

### 4. 마이페이지 (`/mypage`)

Steam 로그인 필수.

**표시 정보:**
- 내 소유 게임 목록
- 게임별 플레이타임

## 인증 플로우

1. 네비게이션에 "AI 추천" 탭 항상 노출
2. 비로그인 상태에서 클릭 → Steam OpenID 로그인 페이지로 리다이렉트
3. 로그인 완료 → Steam Web API로 유저 라이브러리 + 플레이타임 조회
4. AI 추천 페이지 및 마이페이지 접근 가능

---

## 데이터 설계

### 게임 데이터 수집

- **방식**: Steam API 크롤링 → PostgreSQL 저장
- **업데이트 주기**: 48시간마다 배치 작업으로 DB 내 전체 게임 정보 순회 및 최신화
- **수집 항목**: 게임 ID, 제목, 설명, 장르, 태그, 가격, 할인율, 할인 종료일, 리뷰 수, 긍정 리뷰 비율, 스크린샷 URL, 출시일, 시스템 요구사양

### 유저 데이터

- Steam OpenID 인증 후 Steam Web API로 조회
- 소유 게임 목록, 게임별 플레이타임

---

## AI 추천 로직

1. 유저 라이브러리에서 장르별 플레이타임 집계 → 성향 프로파일 생성
2. 유저가 선택한 4가지 조건으로 DB에서 후보군 필터링 (백엔드 사전 필터링 → AI 비용 절감)
3. 필터링된 후보군 + 유저 성향 프로파일을 AI API에 전달
4. AI가 추천도 점수 + Top 3 근거 반환
5. 프론트엔드에서 결과 렌더링

## 네비게이션 구조

```
[로고/홈] [AI 추천] [마이페이지(로그인 후)] [Steam 로그인/프로필]
```

- AI 추천: 항상 노출, 비로그인 시 로그인으로 리다이렉트
- 마이페이지: 로그인 후에만 노출
